Environment variables#

The following environment variables modify the default behavior of both the Python API and the command-line interface. They are intended for setting user preferences.

CJDK_CACHE_DIR#

Set to an absolute path to override the default cache directory, used when not overridden by a keyword argument or command-line option.

CJDK_DEFAULT_VENDOR#

Set to a JDK vendor to override the default (adoptium).
